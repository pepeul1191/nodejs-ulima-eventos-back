## Eventos Ulima Back

Instalar dependencias:

		$ npm install & bower install

Arrancar aplicación JS especificando el puerto:

		$ PORT=8080 nodemon npm start

Arrancar aplicación JS con el puerto por default (3000) seteado en el código:

		$ nodemon npm start
